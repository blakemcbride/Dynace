
Description of files available in this directory.


../README	basic intro, contact information, starting point

INDEX.txt	this file

LICENSE.txt	fully describes what you can and cannot do with this package

DESCRIBE.txt	basic description of the system

ORIENT.txt	description or orientation to all the directories

BUILD.txt	describes how to build the system from scratch

dmake.man	DMAKE (make used under DOS & all Windows) manual

classes		text list of Dynace classes

headers?.txt	discussion of Dynace's header file approach - used as part
		of a discussion on the Dynace mailing list

Various formats of both manuals are located under the "manual" sub-directory.
